Política de Privacidade - Quiz Tech

1. Informações Coletadas

O Quiz Tech coleta as seguintes informações:

  • Nome completo: Usado para identificar tentativas de quiz
  • E-mail: Usado para identificação e possível comprovante
  • Resultados de quiz: Pontuações, respostas e estatísticas
  • Dados de navegação: Páginas visitadas e tempo de uso

2. Como Usamos suas Informações

As informações coletadas são usadas para:

  • Identificar tentativas de quiz
  • Gerar histórico de tentativas
  • Calcular estatísticas de desempenho
  • Melhorar a experiência do usuário

3. Armazenamento de Dados

Os dados são armazenados localmente no seu navegador usando:

  • SessionStorage: Dados temporários da sessão atual
  • LocalStorage: Preferências e configurações
  • IndexedDB: Histórico de tentativas e resultados

Importante: Nenhum dado é enviado para servidores externos. Todos os dados permanecem no seu dispositivo.

4. Compartilhamento de Dados

Não compartilhamos seus dados com terceiros. Todos os dados são armazenados localmente no seu navegador.

5. Seus Direitos

Você tem o direito de:

  • Acessar seus dados armazenados
  • Exportar seus dados
  • Deletar seus dados (limpar histórico)
  • Recusar o uso de cookies (se aplicável)

6. Segurança

Implementamos medidas de segurança para proteger seus dados:

  • Armazenamento local seguro
  • HTTPS para conexões seguras
  • Validação de entrada de dados
  • Prevenção de XSS (Cross-Site Scripting)

7. Cookies

O Quiz Tech pode usar cookies para:

  • Manter preferências do usuário
  • Armazenar configurações de tema
  • Melhorar a experiência de uso

8. Alterações nesta Política

Podemos atualizar esta política de privacidade periodicamente. Recomendamos revisar esta página regularmente.

9. Contato

Para questões sobre esta política de privacidade, entre em contato através do repositório GitHub do projeto.
